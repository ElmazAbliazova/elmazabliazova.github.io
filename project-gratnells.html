Custom Gratnells Bin Storage System

Summer Project for Sears think[box]

Fabricated a Gratnells bin storage unit as part of a summer project for Sears think[box]. The goal was to create a space-efficient solution for organizing tools and materials within the makerspace.

Build Process Video

VCarve

Created 2D layouts and toolpaths using VCarve, optimizing material usage and ensuring clean CNC machinability.

CNC Fabrication

Operated the Laguna CNC router to cut all parts, selecting appropriate feeds, speeds, and tooling for plywood panels.

Wood Shop Post-Processing

Performed sanding and finishing in the wood shop to ensure a clean final aesthetic.
